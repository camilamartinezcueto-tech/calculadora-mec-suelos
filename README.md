# MEC – Mecánica de Suelos (Web Interactiva Educativa)

## Descripción del Proyecto

**MEC** es una aplicación web interactiva desarrollada como herramienta de apoyo académico para estudiantes de Ingeniería Civil en el área de **Mecánica de Suelos**.  

El proyecto permite realizar **simulaciones académicas** orientadas a la comprensión conceptual de fenómenos y parámetros fundamentales del comportamiento de suelos, fortaleciendo el aprendizaje autónomo y la visualización de resultados en un entorno digital accesible.

La aplicación fue concebida como recurso pedagógico para reforzar la interpretación de variables geotécnicas, facilitar la exploración de escenarios y promover el análisis crítico en estudiantes de pregrado.

## Objetivos Académicos

- Apoyar el aprendizaje de conceptos fundamentales en Mecánica de Suelos.
- Facilitar la simulación interactiva de variables geotécnicas.
- Promover el razonamiento técnico y la interpretación gráfica de resultados.
- Integrar herramientas digitales en procesos formativos de Ingeniería Civil.

---

## Alcance del Proyecto

- Aplicación web interactiva orientada a estudiantes.
- Simulaciones académicas de parámetros de suelos.
- No incluye cálculos normativos (ASTM, INVÍAS, NSR-10 u otros marcos regulatorios).
- Uso exclusivamente educativo.

---

## Arquitectura y Tecnologías

El proyecto fue desarrollado utilizando tecnologías web estándar, sin frameworks externos:

- **HTML5** – Estructura de la aplicación  
- **CSS3** – Estilización y diseño visual  
- **JavaScript (Vanilla JS)** – Lógica de simulación e interacción
